Purrfect: Task Companion

Dart Flutter Mobile Dev UI/UX

The Philosophy

Purrfect was born from the idea that productivity tools should be as seamless and "perfect" as possible. I wanted to build an app that balances a high-density data dashboard with a clean, stress-free user interface.

Functional Depth

  • Dynamic Progress Analytics: The "at-a-glance" header provides immediate feedback on completion rates, using custom circular progress indicators and real-time statistics.
  • Integrated Task Calendar: A custom-built bottom sheet calendar that allows for rapid context switching between daily lists and monthly overviews.
  • Smart Lifecycle Management: Tasks are intelligently sorted into Today, This Week, and Later, helping users prioritize what truly matters without manual effort.
  • Visual Craftsmanship: Heavy focus on micro-interactions, soft shadows, and a blue-to-light gradient system that makes the mobile experience feel premium.

Tech Stack Reflection

Developing Purrfect allowed me to master asynchronous data handling in Dart and deep-dive into Flutter's layout engine. The challenge wasn't just making it work—it was making it feel right, from the bounce of a button to the persistence of local data.
