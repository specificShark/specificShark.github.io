Purrfect: Task Companion

Dart Flutter Mobile Dev UI/UX

The Philosophy

Purrfect was born from the idea that productivity tools should be as seamless and "perfect" as possible. I wanted to balance data density with minimalist aesthetics.

Functional Depth

  • Dynamic Analytics: Real-time feedback on task completion rates.
  • Integrated Calendar: Rapid context switching via a custom bottom sheet.
  • Smart Sorting: Intelligent task prioritization (Today/Week/Later).

Tech Stack

Developing this allowed me to master asynchronous data handling in Dart and deep-dive into Flutter's layout engine.
